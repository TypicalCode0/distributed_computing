## 1. Важное уточнение по формату входных данных

В условии задачи присутствовало противоречие:
> *"Файл с массами... на каждой строке 6 дробных чисел (3 координаты и 3 скорости)"*

Для расчета гравитационного взаимодействия по закону всемирного тяготения **необходимо знать массу** каждого тела ($F = G \frac{m_1 m_2}{r^2}$). Поскольку 6 чисел описывают только кинематику ($x, y, z, v_x, v_y, v_z$), **был принят расширенный формат из 7 чисел**.

**Корректный формат строки входного файла:**
```text
m x y z vx vy vz
```
## 2. Порядок работы: Сборка -> Бенчмарки
Для корректного измерения производительности необходимо сначала скомпилировать исполняемый файл из C-кода, создать данные при помощи скрипта build_data.py, и только затем запускать скрипты тестирования или бенчмаркинга.
